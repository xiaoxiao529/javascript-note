可以应用于任意的数据类型 始终返回一个布尔值 对象 ——> false 空字符串 ——> true 非空字符串 ——> false 0 ——> true 非0(包括Infinity) ——> false null ——> true NaN ——> true undefined ——> true 逻辑与&& 可以应用于任意的数据类型 在有一个操作符不是布尔值的情况下，逻辑与操作就不一定会返回布尔值 第一个操作数是对象 ——> 返回第二个操作数 第二个操作数是对象 ——> 只有在第一个操作数的求值结果为true的情况下才会返回该对象 两个操作数都是对象 ——> 返回第二个操作数 有一个操作数是null ——> 返回null 有一个操作数是NaN ——> 返回NaN 有一个操作数是undefined ——> 返回undefined 逻辑与属于短路操作，即如果第一个操作数能够决定结果，那么就不会对第二个操作数进行求值 如果第一个操作数为false，那么无论第二个操作数是什么值，结果都不可能是true了。 逻辑或|| 可以应用于任意的数据类型 在有一个操作符不是布尔值的情况下，逻辑与操作就不一定会返回布尔值 第一个操作数是对象 ——> 返回第一个操作数 有一个操作数的求值结果是false ——> 返回第二个操作数 两个操作数都是对象 ——> 返回第一个操作数 两个操作数是null ——> 返回null 两个操作数是NaN ——> 返回NaN 两个操作数是undefined ——> 返回undefined 逻辑或也属于短路操作，如果第一个操作数的求值结果为true，就不会对第二个操作数求值了。 ps. 我们可以通过逻辑或的这一行为，来避免为变量赋null或undefined值 var myData = preferredData || backupData; 在这个例子中，变量myData将被赋予等号后面两个值中的一个，有且仅有一个 preferredData不等于null，它的值就赋予myData；preferredData等于null，backupData的值就赋予myData。 * 逗号操作符 多用于声明多个变量， var num1=1,num2=2,num3=3; 也可以用于赋值，逗号操作符总会返回表达式的最后一项 var num = (1,2,3,4,0); //num的值是0 函数 代码运行
